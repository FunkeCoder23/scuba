## Code Format
Scuba is compliant with the [Black](https://black.readthedocs.io/)
code style. Code format in PRs is verified by a GitHub action.

To check code formatting:
```
$ ./code_format.py
```

To fix code formatting:
```
$ ./code_format.py --fix
```
